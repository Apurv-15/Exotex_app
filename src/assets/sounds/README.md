# Sound Files

This directory contains sound effect files for the app.

## Required Files:
- `whoosh.mp3` - Whoosh/swoosh sound for form transitions (0.5-1 second)
- `success.mp3` - Success chime for completed actions (0.5-1 second)

## Where to get free sounds:
1. **Mixkit** - https://mixkit.co/free-sound-effects/whoosh/
   - Download "Fast whoosh transition" or similar
   
2. **Freesound** - https://freesound.org/
   - Search for "whoosh" and "success chime"
   - Filter by Creative Commons license

3. **Zapsplat** (free with attribution) - https://www.zapsplat.com/

## Current Status:
The app works perfectly WITHOUT these files using vibration only.
Sound files are completely optional enhancements.
